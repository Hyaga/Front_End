Tags mais usadas para textos basicos

A tag "p" e resposavel pelos paragrafos do HTML

A tag "br" quebra uma linha e a tag "hr" destaca uma linha no html

Tipos de listas no Html

Listas ordenadas(listas que seguem uma ordem ) "ol" e itens nelas sao os "il"

  1. Item 1
  2. Item 2

Listas desordenadas(listas que nescessariamente não seguem um padrão) "ul" os itens sao chamados da mesma forma

  • Item 3
  • Item 4

Tags auxiliares para a formatação do texto

A tag "b" deixa o texto em negrito

A tag "i" deixa o texto em italico

Tag para chamar um link

A tag para colocar um link no html e a tag "a" esta tag tem que esta em um Link, e deve possuir o href para chamar o link
