Titulo

Aqui foi usado uma classe para editar a formatação deste proximo paragrafo, para chamar uma classe e simples

Primeiramente ela deve estar declarada no arquivo css que voce linkou ao html

Depois aonde voce que usala voce deve chama-la na tag exemplo : "p class = "destaque>" e sua tag ira respeitar as regras usadas

Usando a classe destaque

Vale lembrar tambem que para declarar uma classe e nescessario no arquivo css colocar um . antes do nome da classe para que o arquivo entenda que aquilo e uma classe

Exemplo: .destaque, Assim o arquivo entende que destaque e uma classe
