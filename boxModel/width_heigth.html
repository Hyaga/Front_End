Box Model

O box model ou modelo de caixas e resposanvel por ditar as dimenções e um objeto ou seja a largura e altura

Tipos de box model:

  • Box model estatico
  • Box model adaptavel

Aplicação dos tipos de box model

O box model estatico e o box model que o tamanho ja foi definido não mudando com a alteração de tamanho da pagina

O box model adptavel e o box model que muda jutno com a alteração de tamanho da pagina
